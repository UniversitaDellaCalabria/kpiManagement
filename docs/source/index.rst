Gestione e monitoraggio iniziative di Public Engagement
=======================================================

- URL: https://kpi.unical.it/public-engagement-monitoring/;
- Accesso tramite credenziali UniCal ID o SPID/CIE.

**Tipologie di utenti**

- *Referente o delegato:* inserisce dati delle iniziative;
- *Operatore di struttura:*

  - *Validazione dati:* valida le iniziative inserite;
  - *Gestione patrocinio:* gestisce richieste di patrocinio.

- *Operatore di Ateneo:* supervisiona e gestisce dati globali.
